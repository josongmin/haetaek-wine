# 개발 명령어

## 일반 개발 (추천)

```bash
npm run dev
```
- 프론트엔드(3000) + 백엔드(4000) 동시 실행
- HMR 지원으로 빠른 개발
- 프론트에서 API는 proxy로 4000번 포트 호출

## 백엔드만 개발

```bash
npm run dev:server
```
- 서버만 4000번 포트에서 실행
- API 개발/테스트 시 사용

## 프론트엔드만 개발

```bash
npm run dev:client
```
- 클라이언트만 3000번 포트에서 실행
- 백엔드가 이미 실행 중일 때 사용

## 프로덕션 모드 테스트

```bash
npm start
```
- 프론트 빌드 후 4000번 포트에서 통합 실행
- 실제 배포 환경과 동일하게 테스트

```bash
npm run build
npm run serve
```
- 빌드만 따로 하고 서버 실행
- 빌드 후 서버만 재시작할 때 빠름

## 개별 앱 명령어

### 클라이언트
```bash
cd apps/client
npm start        # 개발 서버
npm run build    # 프로덕션 빌드
npm test         # 테스트
```

### 서버
```bash
cd apps/server
npm start        # 서버 실행
```

## 배포

```bash
npm run deploy
```
- 프론트 빌드 + PM2 재시작
